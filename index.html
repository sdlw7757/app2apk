警告：白屏防护

如果安装后看到白屏，请在 Vite 配置中启用 @vitejs/plugin-legacy 插件。

// vite.config.js
import legacy from '@vitejs/plugin-legacy'

export default defineConfig({
  plugins: [
    react(),
    legacy({ targets: ['chrome >= 52'] })
  ],
  base: './'  // Required for APK
})
